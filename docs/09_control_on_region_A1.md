# 09 — Region A_1 control (aligned, not tiny |ω|)

Split A further:
- \(A_1\): \(|\omega|\ge \delta\) (not tiny)
- \(A_2\): \(|\omega|<\delta\) (tiny)

On \(A\), \(|\nabla\xi|\le \sqrt M\) gives local Lipschitz control, and
Constantin–Fefferman depletion implies a subcritical kernel for stretching:
\[
\left|\int_{A_1} \frac{\omega^\top S\omega}{|\omega|^2+\varepsilon^2}\,dx\right|
\le C\sqrt M\,\mathcal S_\varepsilon + C_M\|\omega\|_2^2.
\]
This is absorbable under the NS-PCM dissipation budget.
